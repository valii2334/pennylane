# Pennylane

This is a simple implementation for Pennylane

## Build with

- Ruby on Rails
- Bootstrap with Webpack

## Get Started

### Requirements

- Ruby 2.6
- Rails 6

### Clone

```bash
git clone https://github.com/valii2334/pennylane.git
cd pennylane
```

### Install and migrate db

```bash
bundle install
yarn install --check-files
rails db:create
rails db:migrate
```

### Start Server

To start rails server

```bash
rails server

```
